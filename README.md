# Tracking_Depth_ball_RealSenseL515 - Python

# ====== Description ======

Simple python code using RealSense L515 camera for depth measument.

The ball is find using color detection (OpenCV).

Project demo in youtube:

```bash

```

```bash
pip install matplotlib
```

